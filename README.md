# tudor

CLI application for managing HDR2's platform as a service.

## Usage

```bash
tudor
```

### Configuration

Set your keys with

```
tudor config hdr
```

See currently set keys with

```
tudor config show
```

## Build

To build the binary you'll need [Bun](https://bun.sh) installed. Running `pnpm run build` will output a `tudor` binary.
